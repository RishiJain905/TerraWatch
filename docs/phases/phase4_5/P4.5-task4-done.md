## Filter Count Display

### Globe Info Bar

When no filters are active: `Planes: 10384`
When filters are reducing count: `Planes: 1247 / 10384`

The conditional logic compares `filteredPlanes.length === planes.length` to determine which format to show.

### Sidebar Layer Items

Each layer shows its count next to the label:
- No filters: `✈️ Aircraft  10,384  [toggle]`
- Filters active: `✈️ Aircraft  1,247 / 10,384  [toggle]`

Counts update in real-time as filters change. The count data flows from Globe.jsx through App.jsx to Sidebar via the enhanced `onFilterHooksReady` callback.
