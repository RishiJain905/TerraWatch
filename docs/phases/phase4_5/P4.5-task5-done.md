## ARCHITECTURE.md Changes

Added **"Filtering Architecture"** section covering:

- **Frontend-only filter pattern** — all data flows through existing endpoints/WebSocket channels; filtering is applied in React hooks before reaching the Globe
- **Hook responsibilities** — each hook maintains raw data, filter state, filtered data (useMemo), and updateFilter() function
- **Data flow diagram** — usePlanes/useShips/useEvents/useConflicts → filter functions → Globe layers
- **Filtered data exposure** — Globe uses filtered data for rendering; raw counts shown in info bar
- **Filter UI** — collapsible panels in Sidebar, one per layer; real-time updates; persists across reconnects

## DATA_SOURCES.md Changes

Added **"Filterable Fields"** section documenting per-data-source filter controls:

### Planes (OpenSky Network)
| Field | Type | Filter Control |
|-------|------|----------------|
| `alt` | int (feet) | Altitude range slider (0–50,000 ft) |
| `callsign` | string | Text search (partial, case-insensitive) |
| `speed` | int (knots) | Min speed slider (0–600 kt) |

### Ships (Digitraffic / AISStream.io)
| Field | Type | Filter Control |
|-------|------|----------------|
| `ship_type` | string | Type checkboxes: cargo, tanker, passenger, fishing, other |
| `speed` | float (knots) | Min speed slider (0–30 kt) |

### Events (GDELT)
| Field | Type | Filter Control |
|-------|------|----------------|
| `category` | string | 16 category checkboxes |
| `tone` | float | Tone range dual slider (-10 to +10) |
| `date` | string (YYYY-MM-DD) | Date range dropdown: 24h, 48h, 7d, all |

### Conflicts (GDELT violent subset)
| Field | Type | Filter Control |
|-------|------|----------------|
| `tone` (as intensity) | float | Min intensity slider: weight = \|tone\| + 1 (range 0–11) |
| `date` | string (YYYY-MM-DD) | Date range dropdown: 24h, 48h, 7d, all |

> **Note:** Conflicts are a filtered subset of GDELT events showing only violent/aggressive categories. Intensity is driven by tone magnitude, not fatalities — GDELT does not provide fatality counts.
